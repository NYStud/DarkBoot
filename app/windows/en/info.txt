This application enables the black boot screen on unsupported Macs.[return]
Confirmed working on OS X 10.10 to 10.11[return][return]
Warnings:[return]
On 10.11+ Rootless security must be disabled for Dark Boot to work.[return]
This may break at any time, support for systems below 10.10 is untested.[return]
Use with caution on Beta/Unconfirmed versions of OSX![return][return]
How to use:[return]
• Select your desired Boot Color and press Apply[return]
• Enter your password and press OK[return]
• Reboot twice for changes to take effect[return][return]
How to restore (if you can't boot):[return]
• Boot into your recovery partion (boot holding command+r)[return]
• Open Terminal[return]
• Type in:[return]
	cd /System/Library/CoreServices[return]
	sudo mv ./boot.efi ./boot.efi.bak[return]
	sudo mv ./boot_stock.efi ./boot.efi[return]
	sudo chmod 644 ./boot.efi[return]
	sudo chown root:wheel ./boot.efi[return]
	sudo chflags uchg ./boot.efi[return]
	sudo reboot
